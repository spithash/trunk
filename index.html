This tutorial is written to help TMG administrator to upload URL Sets the easiest way

Steps:

S1-1. Open Forefront TMG Management Console

S1-2. If Task Pane does not appear, Open Task Pane from menu View -> Task Pane

S1-3. On Task Pane, navigate to Toolbox -> Network Objects -> New -> URL Set

S1-4. Create New URL Set, Add Name and dummy URL
 
In this tutorial, set URL Set entry as follows:
  • Name      : Blacklist By IP Address
  • URL Entry : http://DUMMY
 
 
 
S1-5. Right Click on new URL Set Blacklist By IP Address and select Export All
 

S1-6. Open the exported xml file, and what we need is the information that contains the dummy URL (http://DUMMY)

 
S1-8. Open Template file in spreadsheet software and enter URL Entry
 
URL Entry is the address to allow/block
XML Output is the result that we need to paste back on the exported xml.
 
 
S1-9. Highlight the dummy URL entry in exported URL Set xml, replace the xml output (Column C, starting from Cell C3) and save. 

Note: Indentation of pasted xml output is not important.

S1-10. Open the xml file using IE to check saved XML file is valid.

If IE shows error, you have to recheck the xml entry.

S1-11. Return to TMG Console, right click on new URL Sets and select Import All

S1-12. Double click on new URL Set to check the import result.
 
 
S1-13. Add the new URL Sets into Web Access Policy. In this example, the new URL Sets added into Blocked Web Destinations by adding into To Tab
 
 
S1-14. If you are satisfied with the import result, click Apply button to save
